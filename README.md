## Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build process needed — it's pure HTML

### Deploy to GitHub Pages

1. **Create a GitHub repository:**
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git push -u origin main
   ```

2. **Enable GitHub Pages:**
   - Go to Settings → Pages
   - Select "main" branch as source
   - Save

3. **Your site will be live at:** `https://YOUR_USERNAME.github.io/portfolio`

## Customization

### Update Portfolio Info
- Search for "EUGEN MATKOVIĆ" and replace with your name
- Update job title, bio, and descriptions in the hero section
- Modify skills, services, and blog content

### Change Photo
- Replace `uploads/photo-1781820296055.jpeg` with your own photo
- Keep the same filename or update the `src` attributes

### Adjust Colors
The portfolio uses a dark blue/purple gradient. To customize:
- Find hex colors like `#0D1F4C`, `#081529`, etc. and replace
- Update the gradient in the hero section (search `linear-gradient`)

### Update Project Links
- Modify the WORK section project links to point to your projects
- Update card titles and descriptions
